# Student Task & Attendance Management System (Mini Project)

This is a mini frontend project built using HTML, CSS, and JavaScript.

## Project Description
The application allows students to add tasks, track subject-wise work, and mark daily attendance. The main objective of this project is to understand frontend logic, DOM manipulation, and browser-based data storage.

## Features
- Add student tasks
- Track attendance status (Present / Absent)
- Mark tasks as completed
- Delete tasks
- Persistent data using browser localStorage

## Technologies Used
- HTML
- CSS
- JavaScript

## Concepts Learned
- DOM manipulation
- Event handling
- JavaScript arrays and objects
- Data persistence using localStorage

## How to Run
Open index.html in any modern web browser.
